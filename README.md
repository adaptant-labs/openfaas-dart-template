# OpenFaaS Dart Template

An OpenFaaS template for writing serverless functions in Dart.

## Usage

```shell script
$ faas-cli template pull https://github.com/adaptant-labs/openfaas-dart-template
$ faas-cli new hellodart --language dart
```
